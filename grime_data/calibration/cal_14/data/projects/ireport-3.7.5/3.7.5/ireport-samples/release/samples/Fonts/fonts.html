Fonts In iReport you can use all the system fonts you want, but to maximize the portability of the documents we suggest to use TTF Fonts installed using Tools > Options > iReport > Fonts. In this way fonts can be exported as font extensions for JasperReports and deployed as jar in any application.
JasperReports provides a built-in font extension for the DejaVu fonts (Sans, Serif and Mono) which work well with all the character sets and are correctly displayed in PDF documents as well. DejaVu Sans DejaVu Serif DejaVu Sans Mono SansSerif (default font)
