GanttProject - Release 1.9.11

  • PDF/HTML export : fixed bug with '&', '<', '>' symbols in note text.
  • PDF export: added support for Russian (and probably other non-latin-1 compliant) fonts (bug #864770).
  • Resources : fixed invalid mapping of roles to resources after saving file (bug #868946).
  • Task dependencies: added support for task property which indicates if dependency constraints are allowed to shift task backwards even if constraint is formally fulfilled (bug #893139)
  • UI configuration: added fonts customization options (bug #905685)
  • Internationalization: added support for Danish language
  • WebDAV: improved lock reuse (fixes bug #884704), new option: time to live for WebDAV locks
  • MacOS X compatibility: improved selection of LookAndFeels (fixes bug #830947), special startup-script improves integration into the MacOS UI (ganttproject.command)
  • Print: Scale factor on the print preview dialog box.
  • Bug 922246 Tasks' name showing ID.
  • Bug 921804 Task with id -1 and name null appears in the HTML export.
  • Bug 921789 Opening projects generated by previous versions fails.
